# Node-RED IoT con ESP32

Proyecto de control de relays con ESP32 y Node-RED desplegado en Render.

## Características
- Control de relay de 4 canales
- Integración con ESP32
- Red neuronal en Python
- Interfaz web con Node-RED

## Tecnologías
- Node-RED
- Python + TensorFlow
- ESP32
- Docker
